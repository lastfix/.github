# LastFix

개발자 채용 전, 막힌 문제부터 해결합니다.

LastFix는 AI·바이브코딩으로 만든 웹 서비스의 버그, 배포 오류, 운영 이슈를 이어받아 원인을 찾고 필요한 범위만 정확히 수정하는 개발 파트너입니다.

[lastfix.kr](https://lastfix.kr) · [상담 신청](https://lastfix.kr/#contact-form) · [contact@lastfix.kr](mailto:contact@lastfix.kr)

## What We Fix

- Cursor, Bolt, Lovable 등 AI 도구로 만든 코드에서 반복되는 버그
- 로컬에서는 되지만 Vercel, Next.js 배포에서 실패하는 문제
- 결제, 예약, 문의 폼, 관리자 페이지 등 실제 운영 기능의 오류
- 작은 UI 수정, 성능 저하, 비용 증가, 예외 상황 대응
- 외주 프로젝트로 맡기기엔 작고 직접 해결하기엔 어려운 개발 이슈

## How We Work

1. 문의를 남기면 문제 상황과 코드/배포 환경을 확인합니다.
2. 수정 범위, 비용, 일정을 먼저 안내합니다.
3. 착수 후 원인을 분석하고 필요한 변경만 적용합니다.
4. 로컬 확인 결과를 공유한 뒤 운영 반영을 진행합니다.
5. 완료 후에도 재발 가능성이 있는 지점을 함께 정리합니다.

## Services

| 구분 | 내용 |
| --- | --- |
| 단건 버그 수정 | 지금 막힌 오류를 빠르게 확인하고 수정합니다. |
| 배포 오류 해결 | Vercel, Next.js, 빌드 실패, 환경변수 문제를 점검합니다. |

| 운영 개선 | 작은 기능 개선과 유지보수 요청을 현실적인 범위로 처리합니다. |
| 월 구독 파트너 | 반복되는 개발 이슈를 월 단위로 맡기는 방식입니다. 준비 중입니다. |

## Principles

- 문제를 크게 부풀리지 않고 필요한 범위부터 해결합니다.
- 사용자가 이해할 수 있는 언어로 원인과 변경 내용을 설명합니다.
- 정적 페이지, 초기 제품, 소규모 팀의 속도와 비용 감각을 존중합니다.
- 운영 중인 서비스에 적용 가능한 안정적인 수정을 우선합니다.

## Tech Stack

**Frontend**  
React · Next.js (App Router, SSR/SSG) · TypeScript · Tailwind CSS

**Data & Forms**  
TanStack Query · Jotai · React Hook Form · Axios

**Testing & Monitoring**  
Jest · React Testing Library · MSW · Sentry

**Deployment**  
AWS (S3 + CloudFront, 정적 배포 및 캐싱 전략) · Cloudflare Pages · GitHub Actions

**Backend (Light)**  
Node.js · SQL · REST API · Webhook · Auth (JWT/OAuth)

**Integrations**  
결제 연동 · 외부 API 연동
